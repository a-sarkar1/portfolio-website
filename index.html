How to Explore the Map

  • Clusters show the number of photos in each area
  • Click a cluster to zoom in and reveal individual pins
  • Click individual pins to see photo details and images
  • Zoom out to see pins automatically group back into clusters

About This Project

This interactive map visualizes my film photography practice across New York City. Each pin represents a photograph, with data including location, film stock, camera used, and shooting date.

The project demonstrates data collection, geospatial analysis, and interactive visualization techniques—combining my passion for analog photography with data analytics skills.

Technical Details: Built with Leaflet.js for mapping, Marker Cluster for grouping, and CSV data processing. The map updates dynamically as new photographs are added to the dataset.

Data Methodology

Data Collection: Each photograph's metadata was manually cataloged, including precise GPS coordinates obtained through landmark identification and Google Maps verification.

Coordinate Strategy: The map uses dual coordinate systems—neighborhood-level clustering coordinates and precise pin-level coordinates—to balance visual clarity with geographic accuracy.

Ongoing Project: This is a living dataset that grows with each new roll of film. The visualization automatically updates as new data is added to the source spreadsheet.
