About This Project

I wanted to map out my photographic journey in NYC, so this will be a continuous project I will update as I take more photos!

Each pin represents the exact location at which I took a photo. Clicking a pin will allow you to view a few details about the picture, like what camera was used or when it was taken.

I wanted to combine my data analysis knowledge with my passion for film photography and I thought this would be a perfect joining of the two ideas.

My eventual goal with this project is to trace my photographic journey across NYC, capturing candid moments in time to showcase the city that I'm so proud to call my home :)

Data Methodology

Technical Details: Built with Leaflet.js for mapping, Marker Cluster for grouping, and CSV data processing. The map updates dynamically as new photographs are added to the dataset.

Data Collection: Each photograph's metadata was manually cataloged, including precise GPS coordinates obtained through landmark identification and Google Maps verification.

Coordinate Strategy: The map uses dual coordinate systems—neighborhood-level clustering coordinates and precise pin-level coordinates—to balance visual clarity with geographic accuracy.
